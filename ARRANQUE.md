# 🚀 ARRANQUE — Cómo empezar la nueva conversación (a la primera y bien)

## Paso 1 — Instala las skills útiles (en tu terminal de Claude Code, una vez)
✅ **Recomendadas para tu caso** (diseñar webs profesionales):
```
npx -y skills add anthropics/frontend-design --agent claude-code
npx -y skills add nextlevelbuilder/ui-ux-pro-max-skill --agent claude-code
```
🟡 **Opcionales** (temas y artefactos visuales):
```
npx -y skills add anthropics/theme-factory --agent claude-code
npx -y skills add anthropics/web-artifacts-builder --agent claude-code
```
❌ **NO instales** `remotion-video-creation`: es para crear vídeos con código (React); NO encaja con
tus webs HTML autocontenidas y solo añade ruido.

> Nota: las skills mejoran el **diseño**, no hacen magia. Lo que de verdad evita repetir errores es el
> Paso 2 (traspaso) y el Paso 3 (primer mensaje claro).

## Paso 2 — Lo primero que escribes en la conversación nueva
> **"Antes de nada, lee `TRASPASO-INCUBA.md` del repo y respétalo al 100%."**

(Ahí está todo: quién soy, qué quiero, qué hay hecho y los errores que NO debe repetir.)

## Paso 3 — Tu primer mensaje completo (pégalo y rellena lo de [ ])
```
Lee TRASPASO-INCUBA.md y respétalo. Resumen rápido:
- Soy Jaime, de "Incuba tu Negocio".
- Venimos de una conversación donde el Claude anterior se lió: se inventó webs nuevas,
  preguntó de más y no entregó. NO repitas eso.
- Reglas: acción directa, máximo 1 pregunta, no inventes webs, trabaja sobre lo que hay,
  sé honesto desde el principio (un HTML no genera vídeos/fotos), no gastes tokens de más.
- Usa la skill de diseño (frontend-design / ui-ux-pro-max) para que quede profesional.

Mi prioridad AHORA es: [ELIGE UNA → el creador de contenido / automatizar mi Instagram / mi web del estudio]

Hazme UNA sola pregunta para enfocar y luego ve directo a entregar algo concreto. Sin vueltas.
```

## Paso 4 — Si vais a tocar "la web"
Dile que **NO asuma cuál es** y que **NO cree una nueva**. Que te pregunte/enseñe cuál es tu archivo
real de la incubadora "con todo" y lo busque bien antes de tocar nada (esto también está en el traspaso).

---
**Resumen de los 4 pasos:** instala 2 skills → di "lee TRASPASO-INCUBA.md" → pega tu primer mensaje
con tu prioridad → no le dejes inventar webs. Con eso arrancas limpio. 🌱
