# mis-apps · Fábrica de apps con 10 agentes

Sistema para crear webs/apps/embudos de venta de forma **automática**, pasando cada proyecto
por un equipo de **10 agentes especialistas** (arquitecto, marca, UX, copy, frontend, datos,
seguridad, rendimiento, accesibilidad y QA).

## Empieza aquí
👉 Lee **[COMO-USAR.md](COMO-USAR.md)** — guía fácil, sin tecnicismos.

## Contenido
- **`.claude/agents/`** — los 10 agentes especialistas (+ su README).
- **`briefing.html`** — cuestionario para recoger los datos del cliente y generar el encargo.
- **`apps/`** — las apps creadas. Incluye `peluqueria-aurora.html` como ejemplo funcional.
- **`CLAUDE.md`** — instrucciones para que Claude ejecute el pipeline automáticamente.

## Filosofía de cada app
Un solo HTML autocontenido · mobile-first · embudo de venta · sin registro de usuarios ·
panel de admin del dueño · sin recogida de datos personales (sin RGPD) · seguro y rápido.
