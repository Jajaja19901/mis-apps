Invariantes del esquema — activos 24/7 en base de datos ACTIVOS
✅
Trigger trg_contrib_consent_valido
Toda contribución debe ligarse a un consentimiento ACTIVO del mismo usuario. Si no, la BD aborta el INSERT.
✅
CHECK k-anonimato (n_usuarios ≥ 50 y k_aplicado ≥ 50)
Imposible persistir un reporte con menos de 50 usuarios. La BD lo rechaza.
✅
CHECK categorías especiales (es_especial = 0)
